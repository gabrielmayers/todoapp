# To Do App - Django

To Do Web App Using Django!

## Running:

```
git clone https://github.com/gabrielmayers/todoapp.git
cd todoapp
pip install -r requirements.txt
python manage.py runserver

```

## Requisites:

* Python 2.7.17 or Higher
* Django 3.0 or Higher
